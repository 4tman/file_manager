# Консольный файловый менеджер

Двухпанельный консольный файловый менеджер для Windows, написанный на **C++17** с использованием **Windows Console API**. Не требует сторонних библиотек — только стандартный Windows SDK.

---

## Возможности

- Две независимые панели с переключением по `Tab`
- Навигация по файловой системе: вход в папки, выход в родительскую
- Отметка файлов и папок для копирования и перемещения
- Удаление файлов и папок с подтверждением
- Просмотр текстовых файлов с прокруткой
- Два режима отображения: краткий (только имя) и подробный (имя + размер + дата)
- Скачивание файлов по HTTP через встроенный WinINet
- Автоматическое масштабирование под размер окна консоли

## Управление

| Клавиша       | Действие                                              |
|---------------|-------------------------------------------------------|
| `↑` / `↓`    | Переместить курсор вверх / вниз                       |
| `PgUp` / `PgDn` | Прокрутить список на страницу                      |
| `Enter`       | Открыть папку                                         |
| `Backspace`   | Перейти в родительскую директорию                     |
| `Tab`         | Переключить активную панель                           |
| `Пробел`      | Отметить / снять отметку элемента                     |
| `F1`          | Скопировать отмеченный элемент в текущую директорию   |
| `F2`          | Переместить отмеченный элемент в текущую директорию   |
| `F3`          | Просмотр содержимого файла (возврат: `Q` или `Esc`)   |
| `F4`          | Переключить режим отображения (краткий / подробный)   |
| `F5`          | Скачать файл по HTTP-ссылке                           |
| `Del`         | Удалить элемент под курсором (с подтверждением Y/N)   |
| `Q`           | Выход из программы                                    |

**В режиме просмотра файла:**

| Клавиша            | Действие             |
|--------------------|----------------------|
| `↑` / `↓`         | Прокрутка на строку  |
| `PgUp` / `PgDn`   | Прокрутка на страницу|
| `Q` / `Esc`       | Вернуться к списку   |

## Требования

- Windows 10 версии 1903 и выше (Windows 11 поддерживается)
- Компилятор с поддержкой C++17:
  - **Visual Studio 2019 / 2022** (рекомендуется)
  - **MinGW-w64 g++ 12+** (MSYS2)
- Windows SDK (входит в любую установку Visual Studio)

---

## Сборка

### Visual Studio

1. Создать новый проект: **File → New → Project → Console App (C++)**, пустой.
2. Добавить `main.cpp` в проект (Project → Add Existing Item).
3. Установить стандарт C++17:
   > Project → Properties → C/C++ → Language → C++ Language Standard → **ISO C++17 (/std:c++17)**
4. Собрать: **Build → Build Solution** (`F7`).

### Командная строка (MSVC)

Открыть **Developer Command Prompt for VS** и выполнить:

```cmd
cl /std:c++17 /O2 /EHsc main.cpp /link wininet.lib shlwapi.lib
```

### MSYS2 / MinGW-w64

```bash
g++ -std=c++17 -O2 -Wall -o filemanager.exe main.cpp -lwininet -lshlwapi
```

## Запуск

```cmd
filemanager.exe
```

Программа запускается в текущей директории и автоматически разворачивает окно консоли на весь экран. Для завершения нажмите `Q`.

> **Важно:** запускайте в отдельном окне консоли (`cmd.exe` или Windows Terminal), а не во встроенном терминале VS Code — Windows Console API требует полноценного консольного окна.

## Устройство кода

Весь проект — один файл `main.cpp` (~730 строк). Структура по секциям:

| Секция               | Что делает                                                          |
|----------------------|---------------------------------------------------------------------|
| Консольные утилиты   | `gotoXY`, `printFixed`, `printAt`, `clearScreen`, цветовые атрибуты |
| Форматирование       | `formatSize` (B/K/M/G), `formatTime` (YYYY-MM-DD HH:MM)            |
| `struct Entry`       | Один элемент ФС: путь, тип, размер, дата модификации               |
| `struct Panel`       | Директория, список, курсор, прокрутка, режим отображения           |
| Отрисовка            | `drawPanel`, `drawStatusBar`, `drawHelpBar`                         |
| Диалоги              | `inputLine`, `confirm`, `viewFile`                                  |
| HTTP-загрузка        | `downloadHTTP` через WinINet                                        |
| Файловые операции    | `copyRecursive` через `std::filesystem`                             |
| Главный цикл         | Инициализация, `ReadConsoleInputA`, обработка клавиш               |

## Используемые библиотеки

| Библиотека            | Откуда                   | Зачем                                          |
|-----------------------|--------------------------|------------------------------------------------|
| `windows.h`           | Windows SDK (встроен)    | Консольный вывод, ввод клавиш, управление окном |
| `wininet.h`           | Windows SDK (встроен)    | Скачивание файлов по HTTP/HTTPS                |
| `std::filesystem`     | Стандартная C++17        | Обход директорий, копирование, удаление        |

Сторонние библиотеки не используются.
